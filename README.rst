jenkins-github-webhook-plus
===========================

Proxy Service for Sidelaunching Jenkins Tasks on GitHub Events

Usage
-----

::
    python setup.py install
    export JENKINS_URL=http://mylocaljenkins:8080;
    run-jghwhp

or

::
    python setup.py install
    export JENKINS_URL=http://mylocaljenkins:8080;
    python jghwhp
